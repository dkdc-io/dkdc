# dkdc

***Develop knowledge, develop code.***

Command line tools for developer productivity and automation.

## installation

```bash
cargo install dkdc
```

## usage

### config

Open the config file with:

```bash
dkdc config
```

Or:

```bash
dkdc c
```

And save the file.

## open

Open something with:

```bash
dkdc open thing
```

Or:

```bash
dkdc o thing
```

Or:

```bash
dkdc o alias
```

Or:

```bash
dkdc o a
```
